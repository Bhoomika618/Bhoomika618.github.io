About the Event

The Android App Development Workshop 2025 is designed for students and beginners who want to understand the core concepts of Android development. This event covers Android history, SDK setup, application components, architecture, and resource management through interactive sessions and demos.

Key Topics Covered

  • Introduction to Android & History
  • Open Handset Alliance & Android Ecosystem
  • Android SDK Installation and Versions
  • Android OS Architecture
  • Android Virtual Device (AVD)
  • Managing Application Resources
  • Activities, Services & Lifecycles
  • Broadcast Receivers and Content Providers
  • Intents & Intent Filters
  • Android Manifest File
